How SecureNet Works

Multi‑layered security pipeline (project overview)

1

User Attempts Login

Keystroke timing data is captured during password entry, along with network metadata.

2

Behavioral Analysis

Typing patterns are compared against the user's unique baseline profile for deviations.

3

ML Risk Assessment

Deep learning model processes 19 features and predicts risk level with 98.82% accuracy.

4

EDNS Security Check

DNS‑based threat intelligence identifies malicious IPs, VPNs, and TOR exit nodes.

5

Decision Engine

Risk scores are fused: ALLOW, MFA_REQUIRED, or BLOCK based on total threat level.
